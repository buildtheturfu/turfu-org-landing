  <action>
Create an accessible single-select combobox component following WAI-ARIA patterns from the research doc.

**Props interface:**
```typescript
interface ComboboxInputProps {
  id: string;
  label: string;
  value: string;
  options: string[];
  onChange: (value: string) => void;
  placeholder?: string;
  allowCustom?: boolean; // Default true - allows values not in options
}
```

**Required ARIA attributes:**
- Input: role="combobox", aria-controls, aria-expanded, aria-autocomplete="list", aria-activedescendant
- Listbox: role="listbox", aria-label
- Options: role="option", aria-selected, unique IDs matching aria-activedescendant pattern

**Keyboard navigation:**
- ArrowDown: Open dropdown if closed, move to next option if open
- ArrowUp: Move to previous option
- Enter: Select active option, or confirm custom input if allowCustom
- Escape: Close dropdown
- Tab: Close dropdown, let focus move naturally

**Visual styling:**
- Use semantic Tailwind classes: bg-overlay, border-border, text-foreground, focus:border-turfu-accent
- Dropdown: bg-surface-elevated, shadow-lg, max-h-60 overflow-auto
- Active option: bg-turfu-accent/20
- Hover: hover:bg-overlay

**Critical implementation details:**
- Use setTimeout(150ms) in onBlur to allow click events to fire before closing
- Scroll active option into view with scrollIntoView({ block: 'nearest' })
- Filter options case-insensitively
- Sync inputValue with value prop when value changes externally
