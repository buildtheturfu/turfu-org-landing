**Create `src/components/layout/ProseLayout.tsx`** (server component, no 'use client'):

```typescript
interface ProseLayoutProps {
  children: React.ReactNode;
  className?: string;
}

export default function ProseLayout({ children, className = '' }: ProseLayoutProps) {
  return (
    <div className={`max-w-prose mx-auto px-4 sm:px-6 lg:px-8 ${className}`}>
      {children}
    </div>
  );
}
```

- max-w-prose = 720px (from tailwind.config.ts)
- Responsive horizontal padding: 16px mobile, 24px sm, 32px lg
- Accepts optional className for page-specific additions (py-* for vertical spacing)
- Export as default
