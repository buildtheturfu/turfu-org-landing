**Text colors:**
- `text-white` → `text-foreground`

- `bg-white/10` → `bg-overlay-hover`

**Borders:**
- `border-white/5` → `border-border-muted`
- `border-white/10` → `border-border`
- `border-white/20` → `border-border`

**EXCEPTIONS - Keep white text for:**
- Gradient background buttons (btn-primary)
- Accent-colored backgrounds where white provides necessary contrast

Check each file for any hardcoded color values and convert to semantic equivalents.
